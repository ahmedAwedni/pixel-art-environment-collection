# Pixel Art Environment Collection

This repository contains a series of standalone pixel art environment illustrations created as part of my practice in environment design.

## 🎨 Focus Areas

- Composition and scene balance  
- Depth through layering  
- Color harmony  
- Atmospheric lighting  
- Readability at small scale  

## 🖼 Included Environments

- Forest River  
- Coastal Cliff  
- Small Pond  
- Rolling Hills  
- Rocky Valley  
- Desert Path 
- Sakura Tree

## 🎯 Purpose

I've created these pieces as a part of my my environment design skills practice in pixel art and to explore different natural biomes.

I will try to be adding more environments will be added over time.
